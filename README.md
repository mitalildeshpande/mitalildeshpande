## 🌟 Featured Projects

### 📈 [Liquor Demand Forecasting with LSTM](https://ieeexplore.ieee.org/document/9155712) — *IEEE ICESC 2020*
Full-stack ML system forecasting retail liquor sales for inventory optimization. A/B tested MLR vs SVM vs LSTM — LSTM hit **90% accuracy**. Deployed via FastAPI + MongoDB + Kubernetes, handling 1,000+ forecasting requests/day. Results visualized in **Tableau** dashboards.
- **Tech:** Python, Keras, FastAPI, MongoDB, Kubernetes, Tableau

### 🌦️ [Distributed Weather Information System](https://github.com/mitalildeshpande/Weather-Information-System)
Fault-tolerant distributed weather app implementing the **RAFT consensus protocol** across a 5-node cluster for leader election.
- **Tech:** Flask, MongoDB, Docker, RAFT

### 📞 [Telemarketing Analytics for Banks](https://github.com/mitalildeshpande/Telemarketing-analysis-for-Banks)
End-to-end ML pipeline predicting bank term deposit subscriptions from Portuguese telemarketing data — EDA, feature engineering, and classification.
- **Tech:** Python, scikit-learn, Pandas

### 🖼️ [Background Stitching & Image Panorama](https://github.com/mitalildeshpande/Background-Stitching-and-Image-Panorama)
Image stitching pipeline using keypoint detection and homography to merge scenes with moving foregrounds and build seamless panoramas.
- **Tech:** Python, OpenCV, NumPy

### 👤 [Face Detection Benchmark](https://github.com/mitalildeshpande/Face-Detection)
Computer vision pipeline comparing classical (Haar Cascade — 78.91%) vs deep learning (`face_recognition` — 45.1%) approaches for detection and clustering at scale.
- **Tech:** Python, OpenCV, face_recognition

### 🔠 [OCR from Scratch](https://github.com/mitalildeshpande/Optical-Character-Extraction-using-feature-extraction)
Optical character recognition built ground-up using connected component analysis and feature extraction — zero off-the-shelf OCR libraries.
- **Tech:** Python, OpenCV
